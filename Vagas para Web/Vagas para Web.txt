Vagas para Web
==============

1. Visão
--------

Mostrar em uma aba de Facebook todas as vagas de empregos e estágios sem restrições de sexo, idade, e outros.

1.1. Declaração do problema
-------------------------

P1.
O problema é: As ofertas que chegam ao usuário não são relevantes
Afeta: Usuários
Cujo impacto é: Clientes perdem interesse
Uma boa solução seria: Ter uma aba na página mostrando as vagas

1.2. Envolvidos
---------------

A. Programador (Eduardo e Adriano)
B. Cliente (Vagas RS)
c. Usuários do Facebook (alvo)
D. Empresas (Clientes do Vagas RS)
E. Atendimento (?)


Atendimento
 Atende os clientes e cadastras as vagas - Seleciona as vagas que irão aparecer no sistema Vagas para Web.

Usuários do Facebook
 Usuário do Facebook - Visualizar e filtra as vagas no sistema Vagas para Web.

Empresas 
 Empresas que irão fornecer as vagas de emprego ou estágio - Fornecer os dados para atendimento cadastrar as vagas.


1.3. Funções principais
-----------------------

1. Filtro básico - Filtro de por cidade e/ou curso.
2. Ordem da lista de vagas - Ordenação por nome da cidade ou bolça.
3. Detalhes da Vaga - As vagas devem ter as seguintes informações: bolas, turno, código da vaga, benefícios, atividades e observações.
4. Vagas, Cidades e Cursos - Gerenciamentos de Vagas, Cidades e Cursos.

2. Histórias de usuário

Exemplo: As a <type of user>, I want <some goal> so that <some reason>.
